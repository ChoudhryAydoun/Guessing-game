# Number Guessing Game (Python)

A simple interactive number guessing game built using Python.  
The program generates a random number and challenges the user to guess it.

## Features
- Random number generation
- User input handling
- Feedback system (Too High / Too Low)
- Score tracking (optional)

## Technologies Used
- Python

## How to Run

python main.py

## Learning Outcomes
- Learned how to use random module in Python
- Practiced conditional statements and loops
- Improved user input validation and control flow
